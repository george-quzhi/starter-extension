# starter-extension

starter-extension 是基于starter-cli的一个Visual Studio Code的插件，用来生成typescript-express-starter工程代码

starter-cli项目地址：https://github.com/george-quzhi/starter-cli

## Features

### 支持生成以下模块文件
route
controller
dto
model
service
interface
test
